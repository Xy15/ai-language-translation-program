Language Translation Program (OCR + NLP)

Done by a team of 3 members.

Modules:
1. Image Preprocessing
2. Natural Language Processing
3. Optical Character Recognition

Before running:
Do download Tesseract OCR application
Get traineddata from https://github.com/tesseract-ocr/tessdata
E.g. chi_tra.traineddata, eng.traineddata, chi_sim.traineddata
Add "C:\Program Files\Tesseract-OCR\" in Path
1. Open control panel
2. Search "Advance System Settings"
3. View Environment Variable
4. Click on Path and Edit
5. Add "C:\Program Files\Tesseract-OCR\"

Add the trained data files included in the Tesseract folder in "C:\Program Files\Tesseract-OCR\tessdata"

Ensure all installations in the python files have been installed.
